# Bank Loan Analysis Dashboard

## Project Overview

This project analyzes bank loan data using Power BI to provide insights into loan applications, funded amounts, repayments, customer profiles, and loan performance. The dashboard helps identify lending trends, assess risk, and support data-driven decision-making.

## Tools & Technologies

* Power BI
* Excel/CSV
* Data Visualization

## Key Performance Indicators (KPIs)

* Total Loan Applications
* Total Funded Amount
* Total Amount Received
* Average Interest Rate
* Average Debt-to-Income (DTI) Ratio
* Good Loan Percentage
* Bad Loan Percentage

## Dashboard Insights

* Loan applications by month
* Loan status distribution (Fully Paid, Current, Charged Off)
* Funded amount by loan purpose
* Loan analysis by state and region
* Analysis by employee length and home ownership
* Interest rate trends across loan grades
* Good vs Bad loan comparison

## Business Impact

* Helps identify high-risk loan segments
* Tracks loan repayment performance
* Supports better lending decisions
* Improves portfolio monitoring and risk management

## Conclusion

The Bank Loan Analysis Dashboard provides a comprehensive view of lending operations, customer behavior, and loan performance, enabling financial institutions to make informed business decisions.
